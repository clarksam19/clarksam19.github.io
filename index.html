I recently built Lodge, an open source logging observability framework built on top of the Elastic stack and deployed on AWS.

Lodge is an open source self-managed logging observability framework for small to medium-sized microservice applications.

Lodge is built using the Elastic stack (Elasticsearch, Logstash, Kibana, Filebeat) plus Kafka and Amazon S3 for resilence to bursty traffic and cost-efficient data archiving
